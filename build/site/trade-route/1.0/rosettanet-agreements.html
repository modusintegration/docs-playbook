RosettaNet Agreements

Configuring a agreement type for a partner means specifying a Name and

  1. In the CONFIGURATION section of the left-hand navigation pane on the Company Information Page, click RosettaNet Agreements.

    The RosettaNet Agreements Page appears.

Figure 1. RosettaNet Agreements Page

  1. From this page, you can:

    • Edit an Existing Agreement

    • Add a New Agreement

Edit an Existing Agreement

  1. On the RosettaNet Agreements Page, click the Agreement you want to edit.

    The Existing Agreement Page appears.

Figure 2. Existing Agreement Page

  1. Edit settings as appropriate.
    For information about editing specific settings, see Add a New Agreement.

Changing the values for Name or Message Type may disrupt existing processes that refer to the existing values.

Add a New Agreement

  1. Click New. The Agreement Page appears.

Figure 3. Agreement Page

Enter settings as appropriate.
