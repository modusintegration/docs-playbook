Maps

Configuring a map for a partner enables you to specify:

include::/_sources/edit-settings.adoc[]

  1. In the CONFIGURATION section of the left-hand navigation pane on the Company Information Page, click Maps.

    The Maps Page appears.

Figure 1. Maps Page

  1. From this page, you can:

    • Edit an Existing Map

    • Add a New Map

Edit an Existing Map

  1. On the Maps Page, click the map you want to edit.

    The Map Page appears.

Add a New Map

  1. On the Maps Page, click New.

    The Map Page appears.

Figure 2. Map Page
Setting Description

Name

Name for this map.

Required?

No

Default

None

Source Document, Target Document

When you click either the Source Document or the Target Document box, the Pick document selection page appears. You can select from available documents.

Select a mapping type from the dropdown listbox.

  • 'XSLT'
