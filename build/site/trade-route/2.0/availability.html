Availability

You can configure windows (periods of time) during which partners or transactions are on-hold or inactive.

include::/_sources/edit-settings.adoc[]

  1. In the CONFIGURATION section of the left-hand navigation pane on the Company Information Page, click Availability.

    The [img-availability] appears.

Figure 1. Availability Page

To Add a New Partner Status Window

  1. On the [img-on-hold-inactive], in the Partner Status Windows section, click Create New Window.
    The Partner Status Window Page appears.

Figure 2. Partner Status Window Page

  1. On the Partner Status Window Page, in the Type box, click the down-arrow to expose the drop-down list.

  2. From the drop-down list, to create an on-hold window, click ON-HOLD.
    To create an inactive window, click INACTIVE.

  3. Use the calendars and dropdown lists in the Time Bar to identify the start and end dates and times for the window.

  4. Click Save to save the window.

To Add a New Transaction Status Window

  1. On the [img-on-hold-inactive], in the Transaction Status Windows section, click Create New Window.
    The Transaction Status Window Page appears.

Figure 3. Transaction Status Window Page

  1. On the Transaction Status Window Page, in the Type box, click the down-arrow to expose the drop-down list.

  2. From the drop-down list, to create an on-hold window, click ON-HOLD. + To create an inactive window, click INACTIVE.

  3. Use the calendars and dropdown lists in the Time Bar to identify the start and end dates and times for the window.

  4. To add a channel, in the Channel section, click Add Channel.
    The Pick a Channel Window appears.

  5. Select a channel, then click Use selected.

  6. Click Save to save the window.

To Edit an On-Hold or Inactive Window

  1. On the [img-on-hold-inactive], in the section that corresponds to the type of window you want to edit, identify the window you wan to edit and click it.
    The appropriate page appears.

  2. On the page, make changes as appropriate.

  3. Click Save to save your changes.
